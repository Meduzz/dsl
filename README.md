# DSL
Service definition DSL. The DSL focuses on describing what the service provides and what the service expects of it's runtime environment.

The DSL have a number of modules that are opt in. Also inside the modules the data collected is mostly opt in too.

It all starts with an app, from there you add a service. If you have more than one service you keep adding them.
